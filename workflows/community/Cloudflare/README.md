# Cloudflare Hyperautomation Workflows

## Overview
SentinelOne and Cloudflare have partnered to create customized workflows for the SentinelOne Hyperautomation platform which automate and enhance the security of both vendors’ products.  This document outlines how to install and configure these custom workflows. 

Organizations leveraging these workflows can automatically extend endpoint threat intelligence to the network edge. The integrations ensure that threats detected on a single device—whether network scans, identity compromises, or phishing attempts—trigger instantaneous, enterprise-wide defense responses across Cloudflare's Web Application Firewall (WAF), Zero Trust Network Access (ZTNA) and Email Security services.
