# Community Hyperautomation & Response Workflows

This directory contains community-contributed workflows for SentinelOne Singularity.

## Documentation Standard

Workflows are organized by **vendor**, with each workflow in its own versioned subfolder containing:
- `workflow.json` – executable workflow definition
- `README.md` – purpose, Mermaid diagram, use case
- `metadata.yaml` – standardized metadata

This structure enables clear collaboration and visual documentation of complex (including Hyperautomation) workflows.

See the [workflow-template](./workflow-template) for a starting point.
