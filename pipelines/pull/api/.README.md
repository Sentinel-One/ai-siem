# pipelines/pull/api/

Pipelines that **poll a vendor REST/HTTP API** on a schedule.

## Layout

```
pull/api/<vendor>/<product>/
  ├── metadata.yaml
  └── observo_export_pipeline_*.json
```

## Required metadata

```yaml
metadata_details:
  vendor: "<canonical_vendor_key>"
  product: "<canonical_product_key>"
  ingest_mode: "API Call"
  auth_type: "OAuth | Bearer Token | API Key & Secret | Basic | mTLS"
  # ... plus the standard pipeline fields including scheduling.interval_secs
```
